The FIFO_TEST includes a FIFO IP, a 1-port RAM IP and three 2-ports RAM IP cores, 
whose compare_size code compares data that were inputted into the system, outputting 
the max data and address. The other two 2-ports RAM IP cores are named reverse and 
fftshift for outputting two different data array forms.

> *The obsolete code file is code that is discarded because of the long array variable,*
> *which can not be used in the EP4CE10F17C8 registers.*
